# Dependencies
Lib dependencies:
- aws-cli:
    ```
    sudo apt-get install awscli
    aws --version
    aws configure
    ```
- boto3:
    ```
    sudo apt-get install python3-boto3
    pip3 install boto3 --upgrade
    ```
